WHAT IS MARKDOWN

Markdown is a lightweight markup language with plain-text formatting syntax. It was created by John Gruber and Aaron Swartz in 2004 with the goal of allowing people to write using an easy-to-read and easy-to-write plain text format, which can then be converted into structurally valid HTML. Markdown is often used for formatting simple documents, creating documentation, writing readme files, and formatting text on web platforms like GitHub, Reddit, and Stack Overflow. Markdown works by using simple symbols and conventions to indicate formatting elements.
